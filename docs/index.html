Caso de estudio

E.E.D.A. — Ecosistema Educativo Digital Adaptable

Plataforma educativa de doble entorno (Android + escritorio) para ciberseguridad y lógica computacional. Orientada a niños y adolescentes de 3 a 16 años bajo el modelo pedagógico CPA (Concreto → Pictórico → Abstracto).

📱 E.E.D.A. Android

Aplicación móvil nativa

Aplicación completa en Jetpack Compose con Material Design 3 sobre Clean Architecture. Sistema de gamificación con el "Perfil del Aventurero" — métricas en tiempo real de Lógica, Creatividad y Resolución de Problemas. Rendimiento optimizado para dispositivos de gama baja mediante carga diferida y StateFlow.

Jetpack Compose Material Design 3 Coroutines / Flow Clean Architecture UI Adaptativa
💻 E.E.D.A. Escritorio

Framework modular de escritorio

Versión desktop con Python y Flet bajo arquitectura modular propia. Dos motores especializados: ContentEngine (distribución progresiva de contenido) y DatabaseManager (persistencia local con integridad transaccional). 8 unidades temáticas autónomas con funcionamiento offline.

Python 3.12 Flet v0.27 SQLite3 ContentEngine Offline-First

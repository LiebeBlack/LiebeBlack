# 🐺 LiebeBlack Portfolio & Github Documents

Bienvenido al repositorio documental y la Masterpiece Web de **Yoangel Gómez (alias Liebe Black)**.

## 🎯 Sobre este Proyecto
Este proyecto fue diseñado específicamente para servir como una **página web maestra, hermosa y premium**, diseñada como perfil personal y profesional para ** GitHub**. Se ha construido con un enfoque de alto nivel, utilizando **HTML puro, CSS Vanilla y JavaScript**, logrando un performance excelente y una estética Cyber-Elegante en donde destacan el Glassmorphism y las interacciones minimalistas de alto impacto.

## 🚀 Arquitectura y Tecnologías
- **HTML5 Semántico**: Para una estructura clara y un SEO robusto.
- **CSS Vanilla (Custom Properties)**: Variables, flexbox, grid, animaciones 60fps (glitch effects, parallax, hover glows, custom cursor).
- **JavaScript (ES6+)**: Interactividad del DOM, Intersection Observers para animaciones al scroll y manejo del custom cursor.
